Storage pattern simulations

Forklift storage logic — each square is a stack of 3 bales, each bay holds 12 stacks.
Rule: every stack goes to the leftmost bay with the fewest stacks. This produces tiered level-filling: the shallowest bays are raised to match the next tier, always sweeping left-to-right. Press Start — loads then arrive automatically until the zone is full.

Simulation 2 starts with an empty bay D between C and a full E — note the rule sends the forklift into that two-sided corridor first, which is the operator-safety scenario raised in the storage logic thread (scenario 4).
